1. Finalidade desta Política

A presente Política de Privacidade descreve como o Dom Bistrô Grill recolhe, utiliza e protege os dados pessoais dos utilizadores do website https://dombistrogrill.netlify.app/, em conformidade com o Regulamento Geral sobre a Proteção de Dados (RGPD).

2. Dados Pessoais Recolhidos

Podemos recolher os seguintes dados:

  • Nome e apelido
  • Número de telefone
  • Endereço de email
  • Morada de entrega
  • Informações relacionadas com encomendas

3. Como Utilizamos os Dados

Os dados recolhidos são utilizados para:

  • Processar encomendas e entregas
  • Comunicar com o cliente sobre pedidos
  • Emitir faturas e cumprir obrigações legais
  • Melhorar a experiência do utilizador

4. Partilha de Dados

O Dom Bistrô Grill não vende nem partilha dados pessoais com terceiros, exceto quando necessário para:

  • Processamento de pagamentos (ex: EasyPay)
  • Cumprimento de obrigações legais

5. Segurança da Informação

Adotamos medidas técnicas e organizativas adequadas para proteger os dados pessoais contra acesso não autorizado, perda ou uso indevido.

6. Direitos do Utilizador

O utilizador tem o direito de:

  • Aceder aos seus dados pessoais
  • Solicitar correção ou atualização

  • Retirar o consentimento a qualquer momento

O website pode utilizar cookies essenciais para o seu funcionamento. O utilizador pode configurar o seu navegador para recusar cookies.
